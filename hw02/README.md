# EDA: World Cities Population

В данном задании был проведён EDA датасета с информацией о городах мира. В исходных данных содержатся сведения о стране, регионе, названии города, его географических координатах и численности населения.
В процессе работы была выполнена очистка данных, например, были удалены дубликаты, а также рассмотрены различные подходы к обработке пропусков. Основной анализ проводился на датасете без пропущенных значений. 
Было исследовано распределение численности населения городов с помощью гистограмм и boxplot. Установлено, что распределение имеет выраженную асимметрию вправо, что связано с большим количеством малых городов и наличием крупных мегаполисов. Также были выявлены выбросы и применены методы их фильтрации. Для улучшения визуализации использовалось логарифмирование переменной Population.
Были построены различные визуализации, включая scatterplot для анализа географического распределения городов, countplot для оценки распределения по странам, а также heatmap и pairplot для анализа взаимосвязей между признаками. Корреляционный анализ показал, что линейная зависимость между числовыми признаками выражена слабо.
Дополнительно было выполнено категориальное разбиение городов по численности населения, которое показало, что в датасете преобладают малые и средние города, тогда как крупные встречаются значительно реже. Также были применены базовые методы feature engineering и кодирования категориальных признаков.
